# Algorithm-Checkpoint-01-YB
Introduction to Algorithm and basic concept 

Sentence Analysis:

This simple JavaScript program analyzes a given sentence to provide the following 

information:

The length of the sentence (the number of characters).
The number of words in the sentence (assuming that the words are separated by a single space).
The number of vowels in the sentence.

Usage:

To use this program, simply run the provided JavaScript code. It will output the analysis results to the 
console.

Evaluation:

This code meets the requirements of the task by using three variables as counters and providing the requested
analysis of the sentence.
